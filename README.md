# SergioAndrade22-servicio-web

Servicio creado utilizando nodejs express con npm como manejador de paquetes

## Dependencias utilizadas:
* [express](https://www.npmjs.com/package/express): framework de aplicación utilizado para desarrollar el server
* [nodemon](https://www.npmjs.com/package/nodemon): utilizado durante el desarrollo para correr el server
* [dotev](https://www.npmjs.com/package/dotenv): utilizado para almacenar la información sensible del sitio y poder referenciarla en el código
* [pg](https://www.npmjs.com/package/pg): utilizado para manipular la base de datos postgresql de [Heroku](https://familysize.herokuapp.com/)
